    25 Comandos Básicos (Sistema)
    • cd Entra ou sai de uma pasta (ex: cd documentos).
    • rmdir Apaga uma pasta que esteja vazia.
    • rm Apaga um arquivo definitivamente.
    • cp Copia um arquivo para outro lugar.
    • mv Move ou troca o nome de um arquivo.
    • touch Cria um arquivo em branco.
    • cat Mostra todo o texto que está dentro de um arquivo.
    • less Abre o arquivo para leitura permitindo rolar a tela.
    • head Mostra apenas as primeiras linhas do arquivo.
    • tail Mostra apenas o final do arquivo.
    • grep Procura uma palavra específica dentro do arquivo.
    • find Faz uma pesquisa por arquivos no computador.
    • chmod Muda as permissões (quem pode ler ou editar).
    • chown Muda o usuário dono do arquivo.
    • man Abre o manual ensinando a usar determinado comando.
    • history Mostra a lista de tudo que você já digitou.
    • sudo Executa o comando com poderes de administrador.
    • apt Baixa, instala e atualiza programas.
    • tar Junta vários arquivos num pacote só (.tar).
    • gzip Comprime o arquivo para ele ocupar menos espaço.
    • unzip Extrai arquivos de um pacote .zip.
    • df Mostra o espaço livre e ocupado no HD.
    • du Mostra o tamanho exato de uma pasta ou arquivo.
    • top Mostra o uso do Processador e Memória RAM ao vivo.
    • ps Mostra os programas rodando no fundo.
    10 Comandos de Rede
    • ping Testa se a internet está funcionando conversando com um site.
    • ip Mostra configurações de rede e o seu endereço IP.
    • ss Mostra conexões de rede e portas abertas no PC.
    • traceroute Rastreia a rota da sua internet até chegar no site.
    • curl Baixa informações e códigos HTML de um site.
    • wget Faz o download de um arquivo da internet.
    • nslookup Descobre qual é o endereço IP escondido num site.
    • dig Faz uma pesquisa técnica detalhada nos servidores DNS.
    • ssh Conecta de forma segura para acessar outro PC de longe.
    • ufw Controla o Firewall (sistema de segurança e bloqueio).
